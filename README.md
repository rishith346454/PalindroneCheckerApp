# PalindroneCheckerApp
    Project Planning

Defined the objective of the application:
To create a program that checks whether a given word or sentence is a palindrome.

Identified key requirements:

Accept user input

Ignore spaces and capitalization

Display whether the input is a palindrome or not

----------------------------------------------------------

Project Setup

Created a new Java project named PalindromeCheckerApp.

Created the main class file PalindromeCheckerApp.java.

Structured the project with proper comments describing:

Running procedure

Flow of the project

Objective

Use cases

---------------------------------------------------------------

Designing the Program Logic

Planned the algorithm:

Take input from the user.

Convert the input to lowercase.

Remove spaces and special characters (if required).

Reverse the string.

Compare the original cleaned string with the reversed string.

Display the result.

--------------------------------------------------

Implementation

Used Scanner class to read user input.

Used string methods such as:

toLowerCase()

replaceAll() (optional for removing spaces)

Implemented string reversal using:

StringBuilder.reverse()
or

A loop-based reversal method.

Added conditional statement (if-else) to check palindrome condition.
